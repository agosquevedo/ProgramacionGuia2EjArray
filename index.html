Guía de Ejercicios 2 - Funciones de Array

Duplicar Números

Se ejecutó la función para duplicar cada número en el array utilizando el método map(). Se espera un nuevo array con los números duplicados.

Ejemplo:

Array original: [1, 2, 3, 4, 5]

Array duplicado: [2, 4, 6, 8, 10]

Números Pares

Se ejecutó la función para filtrar solo los números pares en el array utilizando el método filter(). Se espera un nuevo array con los números pares.

Ejemplo:

Array original: [1, 2, 3, 4, 5]

Números pares: [2, 4]

Libro por Autor

Se ejecutó la función para encontrar un libro con un autor específico utilizando el método find(). Se espera un objeto representando el libro encontrado.

Ejemplo:

Libros:

- Título: "1984", Autor: "George Orwell"

- Título: "Cien años de soledad", Autor: "Gabriel García Márquez"

Libro encontrado: Título: "1984", Autor: "George Orwell"

Persona por Edad

Se ejecutó la función para generar un nuevo array con los nombres de las personas en mayúsculas, filtrar solo las personas mayores de 18 años y encontrar la primera persona que tenga exactamente 25 años utilizando los métodos map(), filter() y find() respectivamente. Se espera un objeto representando la persona encontrada.

Ejemplo:

Personas:

- Nombre: "Juan", Edad: 25

- Nombre: "María", Edad: 30

Persona encontrada: Nombre: "Juan", Edad: 25

Productos con Descuento

Se ejecutó la función para calcular el precio final de cada producto después de aplicar el descuento, filtrar solo los productos con un precio final mayor que 50 y encontrar el primer producto que tenga un descuento del 20% utilizando los métodos map(), filter() y find() respectivamente. Se espera un objeto representando el producto encontrado.

Ejemplo:

Productos:

- Producto: Camisa, Precio: $30, Descuento: 10%

- Producto: Pantalón, Precio: $50, Descuento: 15%

Producto encontrado: Producto: Zapatos, Precio: $80, Descuento: 20%
